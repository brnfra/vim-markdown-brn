# vim-markdown
Auto complete comands to turn easy write markdown files.

## Install

Plug manager;
Ex:

```Plug 'brnfra\vim-markdown-brn'```

## Usage

The most of commands ends with dot. Described bellow;
 
| Command | Write it | 
| --- | --- | 
| ` | \``` ``` | 
| *. | ** | 
| **. | **** | 
| _. | __ | 
| __. | ____ | 
| ~. | ~~~~ | 
| +. | ++++ | 
| =. | ==== | 
| -. | --- | 
| <. | \<!----> | 
| \. | [table2x2] | 
| \n | \<br> | 
| !i | ![ImageDescription](link_to_image) |
| !l | [LinkDescription](link_address) | 
| [^ | [^] | 
| :. | :::::: | 
| \& | &amp;  | 
| \< | &lt; | 
| \> | &gt; | 
| \+ | &nbsp; |
